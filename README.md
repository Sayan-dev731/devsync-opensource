## 🌟 What Makes DevSync Special?

DevSync isn't just another open-source platform – it's a **complete ecosystem** designed for modern developers who want to make meaningful contributions while building their careers. Perfect for **hackathons**, **coding competitions**, and **long-term open-source development**.

### 🎯 **For Contributors**
- 🔍 **Smart Project Discovery**: Find projects that match your skills and interests
- 📊 **Real-Time Progress Tracking**: Watch your contributions turn into achievements

- 🎖️ **Progressive Badge System**: From "Seeker" to "Luminary" - unlock achievements as you grow
- 📈 **Portfolio Building**: Showcase your open-source journey

- 📋 **Project Management**: Easy project submission and approval workflow

- 🎮 **Gamification Engine**: Built-in leaderboards and achievement systems

## 🚀 **Quick Start** - Get Running in 3 Minutes!

### 📋 **Prerequisites**
- Node.js (v16 or higher)
- MongoDB (local or cloud instance)
- GitHub OAuth App (for authentication)
- Gmail account (for email notifications)

### ⚡ **Installation**

```bash
# 🔥 Clone the powerhouse
git clone https://github.com/Sayan-dev731/devsync-opensource.git
cd devsync-opensource

# 📦 Install dependencies (lightning fast!)
npm install

# 🔧 Configure your environment
cp .env.example .env
# Edit .env with your credentials (GitHub OAuth, MongoDB, Gmail)

# 🚀 Launch DevSync
npm run dev
```

### 🌍 **Environment Setup**

Create your `.env` file with these essential variables:

```env
# Database
MONGODB_URI=mongodb://localhost:27017/devsync

# GitHub OAuth (Create app at: https://github.com/settings/applications/new)
GITHUB_CLIENT_ID=your_github_client_id
GITHUB_CLIENT_SECRET=your_github_client_secret

# Email Service
GMAIL_USER=your-email@gmail.com
GMAIL_APP_PASSWORD=your-app-password

# Security
SESSION_SECRET=your-super-secret-session-key
API_KEY=your-api-key-for-admin-access

# Optional: Admin Configuration
ADMIN_GITHUB_IDS=your-github-id,admin2-github-id
```

### 🎯 **First Steps After Setup**
1. **Visit** `http://localhost:3000`
2. **Login** with your GitHub account
3. **Explore** the dashboard and leaderboard
4. **Submit** your first project
5. **Start** contributing and earning points!
